# ksail-docs

Documentation for KSail published as GitHub Markdown files and as a Just the Docs GitHub Pages site.

## Documentation Structure

- **[Overview](docs/overview/)** – Product overview, architecture, project structure, and support matrix
- **[Configuration](docs/configuration/)** – CLI flag reference, declarative configuration guides, and precedence rules
- **[Use Cases](docs/use-cases/)** – Scenario playbooks covering learning, local development, and CI/CD pipelines

## Getting Started

Start with the [documentation README](docs/README.md) for the complete directory layout and navigation guide.
